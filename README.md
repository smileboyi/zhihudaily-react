# 本项目是一个由create-react-app创建的ts版本的简易仿知乎日报程序的web版项目

## 技术栈：
react hooks+typescript+react-router-dom+better-scroll。

## 启动项目：

```
# 克隆项目
git clone https://github.com/smileboyi/zhihudaily-react.git

# 安装依赖
cd zhihudaily-react && npm install

# 运行项目
npm start
```
